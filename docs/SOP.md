# 荷兰语智能记忆系统 - 产品 SOP

## 1. 产品定位

| 项目 | 描述 |
|------|------|
| **名称** | 荷兰语智能记忆系统（Dutch Smart Memory） |
| **类型** | 手机 App（iOS / Android） |
| **核心** | 基于动态难度模型与记忆曲线算法的智能单词学习系统 |
| **目标** | 每天 10-15 分钟高效掌握荷兰语单词 |

---

## 2. 核心功能

### 2.1 学习模块
1. ✅ 每天学习 5-10 个新单词
2. ✅ 自动安排复习（动态难度模型，非固定艾宾浩斯）
3. ✅ 动态评估记忆强度
4. ✅ 根据遗忘风险智能推送复习
5. ✅ 连续 5 天答对才算"真正掌握"

### 2.2 测试形式

| 测试类型 | 描述 | 难度 |
|----------|------|------|
| **选择题** | 看荷兰语 → 选中文 | ⭐ |
| **拼写题** | 看中文 → 输入荷兰语 | ⭐⭐ |
| **听写题** | 语音播放后拼写荷兰语 | ⭐⭐⭐ |

### 2.3 用户自评机制

每次测试后选择：
- ✅ **记住** - 清晰记得，无需思考
- ⚠️ **模糊** - 需要回忆，不太确定
- ❌ **忘记** - 完全想不起来

## 4. 记忆算法

### 4.1 记忆强度更新规则

| 答题结果 | 自评 | 强度变化 | 间隔变化 |
|----------|------|----------|----------|
| ✅ 正确 | 记住 | +8 ~ +12 | ×1.5 |
| ✅ 正确 | 模糊 | +3 ~ +5 | ×1.2 |
| ❌ 错误 | - | -10 ~ -15 | 重置为 4h |
| - | 忘记 | -10 ~ -15 | 重置为 4h |

### 4.2 强度更新公式

```
新强度 = min(100, max(0, 当前强度 + 基础增量 × 随机因子 / 难度系数))

随机因子 = 0.8 ~ 1.2（增加自然波动）
```

### 4.3 间隔计算公式

```
新间隔 = 当前间隔 × 倍数因子 × (记忆强度 / 100 + 0.5)

最小间隔: 4 小时
最大间隔: 30 天
```

### 4.4 难度系数调整

```
答错时: 难度系数 = min(3.0, 当前系数 + 0.2)
连续3次答对: 难度系数 = max(1.0, 当前系数 - 0.1)
```

## 5. 掌握判定

### 5.1 掌握条件

单词被标记为"真正掌握"需同时满足：

1. ✅ 连续 5 天答对（不同日期）
2. ✅ 记忆强度 ≥ 85

### 5.2 状态流转

```
新词(new) 
  → 学习中(learning): 第一次学习后
  → 复习中(reviewing): 记忆强度 > 50
  → 已掌握(mastered): 满足掌握条件
```

### 5.3 低频复习池

已掌握的单词进入低频复习池：
- 复习间隔: 7-30 天
- 如果再次答错，重新进入 `reviewing` 状态

## 6. 每日学习结构

### 6.1 学习比例

| 类型 | 占比 | 说明 |
|------|------|------|
| 新词 | 40% | 当日新学习的单词 |
| 复习词 | 60% | 到期需要复习的单词 |

### 6.2 学习顺序规则

1. 新词与复习词交替出现
2. 避免同类型单词连续出现
3. 高遗忘风险单词优先复习
4. 每 3-4 个单词穿插一个新词

### 6.3 每日目标

- 新词: 5-10 个（用户可设置）
- 复习词: 根据到期数量自动调整
- 建议总时长: 10-15 分钟
